== Description ==

Used as an add-on for mam/linkbuilding plugin to auto update the resources metrics using SEO CHECKER API
== Installation ==

* Upload ”mam-lb-seo-checker” to the ”/wp-content/plugins/” directory.
* Activate the plugin through the ”Plugins” menu in WordPress.
* Add the “MAM Linkbuilding SEO CHECKER ADD ON” to the page where you want to show.
